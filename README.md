# LABH LITE

**Premium Stock Market Trading Simulation Dashboard**

*A sleek, immersive, and visually stunning trading experience built with modern fintech aesthetics.*

</div>

---

### ✨ Features

- Premium **Glassmorphism** UI with vibrant gradients and smooth animations
- Real-time market simulation with live price tracking
- Complete trading system: Order Book, Buy/Sell Ticket, Portfolio & Trade History
- Interactive **3D Global Market Globe** using globe.gl
- AI-Powered Insights powered by **Groq API**
- Secure login and data persistence

### 🚀 Quick Start

```bash
git clone https://github.com/ASHIT2007/STOCK-MARKET-DASHBOARD-L.A.B.H.-
cd STOCK-MARKET-DASHBOARD-L.A.B.H.-
npm install
cp .env.example .env
npm run dev
